Searching Algorithm Visualizer

This is a Project which allows you to visualize how various searching Algorithms work.

Its main purpose is to make the user understand how these Algorithms approch a given element.

Enter the numbers you want OR leave blank for Random values (Input should be Space Seperated Integers with preferable range of 0 to 100 - For Better Visualization)
  • For Depth First Search, the Graph is automatically created with the given input (or Randomely), i.e, the Edges are automatically created.
  • For Hash Map, the values which are smaller than 100 are considered.
